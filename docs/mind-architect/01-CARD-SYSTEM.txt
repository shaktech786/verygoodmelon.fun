                        CARD ANATOMY
================================================================================

Every card has the following properties:

│  WEIGHT: 3    MULTIPLIER: --        │

│  ─────────────────────────────────  │

PROPERTIES EXPLAINED:

COST (Thought Points - TP)
- 0-3 TP per card
- You have 3 TP per turn (can be modified)
- Unspent TP does NOT carry over

WEIGHT (Base Damage)
- Only on Evidence cards
- Range: 1-8
- Represents argument strength

MULTIPLIER
- Only on Logic cards
- Range: 1.2x - 3.0x
- Applied to total weight in chain

RARITY
- Common (★☆☆): 60% of card pool
- Uncommon (★★☆): 30% of card pool
- Rare (★★★): 10% of card pool

SCHOOL
- Which school of thought favors this card
- Affects starting deck composition


================================================================================
                       CARD CATEGORIES
================================================================================

There are 5 main card categories:

1. EVIDENCE CARDS (Blue) - Base damage dealers
2. LOGIC CARDS (Yellow) - Multipliers and modifiers
3. FRAMEWORK CARDS (Green) - Structural/combo enablers
4. META CARDS (Purple) - Special abilities
5. FLAW CARDS (Red) - Negative cards gained from enemies

================================================================================
                    CATEGORY 5: FLAW CARDS
================================================================================

Flaw cards are NEGATIVE cards added to your deck by enemies.
They represent corrupted thinking that weakens your arguments.

You cannot remove Flaws normally - only through special means.

LOGICAL FALLACIES
─────────────────
┌────────────────────────────────────────────────────────────────────────────┐
│ NAME              │ EFFECT                                                 │
├────────────────────────────────────────────────────────────────────────────┤
│ Ad Hominem        │ -2 weight when played. Cannot be discarded.            │
│ Straw Man         │ Distorts your next Evidence card.                      │
│ False Dilemma     │ Can only play 2 cards next turn.                       │
│ Slippery Slope    │ Each card costs +1 TP this turn.                       │
│ Circular Reason   │ This card keeps returning to your hand.                │
│ Appeal to Emotion │ Your Logic cards lose 0.5x multiplier.                 │
│ Hasty General.    │ Your Inductive cards have 50% fail chance.             │
│ Post Hoc          │ Causal cards deal 0 damage.                            │
│ Equivocation      │ One random card changes meaning (bad).                 │
│ Begging Question  │ Your Conclusion cards require 2 Premises.              │
└────────────────────────────────────────────────────────────────────────────┘

REMOVING FLAWS:
- "Clarity" relic: Remove 1 Flaw per rest site
- "Reductio" card: Triple damage if enemy has Flaw (transfers it)
- "Self-Correction" event: Pay coherence to remove Flaws
- Boss defeat: Remove all Flaws gained that floor


================================================================================
                        SYNERGY EXAMPLES
================================================================================

BASIC COMBO (Weight + Multiplier):
Observation (3) + Modus Ponens (1.5x) = 4.5 → 5 damage

CHAIN COMBO (Multiple multipliers):
Statistics (5) + Pattern Matching (1.3x) + Syllogism (2.0x) = 13 damage

SYNTHESIS COMBO (Dialectic):
Thesis + "Free Will" + Antithesis + "Determinism" = Synthesis "Compatibilism"
Base effect: 3.0x multiplier + new card added to deck

STEELMAN COMBO (Meta play):
Copy enemy's Confirmation Bias → Use against them → 2x damage

SOCRATIC CHAIN:
Why? (1.5x) + Why? (2.0x) + Why? (2.5x) + Why? (3.0x) + Why? (3.5x) = 3.5x
"The Five Whys" achievement unlocked


================================================================================
                       CARD UPGRADE SYSTEM
================================================================================

At Rest Sites, you can upgrade cards:

UPGRADE TYPES:
1. ENHANCE: +1 weight or +0.3x multiplier
2. REDUCE COST: -1 TP cost (minimum 0)
3. SPECIALIZE: Add conditional bonus (+2 if...)
4. INNATE: Card always in opening hand
5. RETAIN: Card stays in hand between turns

VISUAL INDICATOR:
- Upgraded cards have golden border
- Card name becomes "Enhanced [Name]"
