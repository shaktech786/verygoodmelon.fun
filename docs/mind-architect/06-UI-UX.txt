================================================================================
                       DESIGN PHILOSOPHY
================================================================================

CORE PRINCIPLES:

1. CLARITY OVER CLEVERNESS
   - Every piece of information is immediately readable
   - No hidden mechanics or obscured data
   - Tooltips available for everything

2. CALM AESTHETICS
   - Paper-craft, hand-drawn feeling
   - Warm, muted colors
   - Plenty of whitespace
   - Reduce visual anxiety

3. ACCESSIBLE BY DEFAULT
   - Keyboard-only navigation
   - Screen reader support
   - Color-blind friendly
   - Adjustable text size

4. RESPONSIVE DESIGN
   - Works on desktop (primary)
   - Works on tablet
   - Works on mobile (simplified)

================================================================================
                    ACCESSIBILITY FEATURES
================================================================================

KEYBOARD NAVIGATION:
```
TAB / SHIFT+TAB    - Move between elements
ENTER / SPACE      - Select/activate
ARROW KEYS         - Navigate cards in hand
1-9                - Quick-select card by position
E                  - End turn
U                  - Undo last action
ESC                - Open menu / Cancel
D                  - View deck
R                  - View relics
?                  - Help overlay
```

SCREEN READER:
- All elements have aria-labels
- Card effects read aloud
- Battle state announced
- Turn changes announced
- Damage and effects vocalized

VISUAL ACCESSIBILITY:
- High contrast mode available
- Color-blind patterns on card types
- No color-only information
- Text scales up to 200%
- Reduced motion option

COGNITIVE ACCESSIBILITY:
- Pause anytime (turn-based)
- Undo available
- Tooltips explain everything
- Tutorial replayable
- "Explain this" button on complex effects


================================================================================
                      ANIMATION PRINCIPLES
================================================================================

TIMING:
- Micro-interactions: 100-200ms
- Card movements: 200-300ms
- Damage numbers: 400ms
- Major transitions: 500ms

EASING:
- Most transitions: ease-out
- Bouncy elements: spring easing
- Damage: sharp ease-in

REDUCED MOTION:
When prefers-reduced-motion is enabled:
- No card flying animations
- Instant transitions
- Static damage numbers
- No particle effects

ANIMATION CATALOG:
- Card draw: Slide up from deck
- Card play: Fly to play area
- Damage dealt: Number pops with shake
- Coherence loss: Bar pulses red
- Enemy defeat: Dissolve effect
- Turn end: Subtle pulse
